# AtlasTime Product Definition

## Problem

People coordinating internationally repeatedly calculate local times, accidentally propose inconvenient hours, and then switch to another app to start the call.

## Value proposition

AtlasTime shows when everyone is available and provides the fastest path from “what time works?” to “start the call.”

## Primary user

A person who regularly communicates with family, clients or colleagues in two or more time zones.

## MVP success criteria

A first-time user should be able to:

1. Add three people in different time zones.
2. Understand their current local times immediately.
3. Select a date and find a reasonable shared hour.
4. launch a preferred communication app.
5. Return later and find the roster preserved.

## Explicit non-goals for MVP

- Full calendar replacement
- Automatic contact harvesting
- Social network
- Video calling infrastructure
- Complex account administration
- AI-generated schedules requiring a backend

## Design principle

Keep the central experience visible on one screen and do not add complexity before the scheduling loop is validated.
